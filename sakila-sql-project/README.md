# Sakila Movie Rental SQL Project

A SQL practice project using the popular Sakila sample database.  
The project focuses on querying films, customers, rentals, categories, and analysing real business-style questions.

## Key Skills Used
- MySQL
- Joins (INNER, LEFT)
- Subqueries
- Group By & Aggregate Functions
- Filtering with WHERE & LIKE
- Ranking & business insights from raw data
