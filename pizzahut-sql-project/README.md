# PizzaHut SQL Sales Analysis Project

A complete SQL-based analysis of a real-world pizza sales dataset.  
This project focuses on understanding sales performance, revenue trends, and customer behaviour.

## Key Skills Used
- MySQL
- Joins & Subqueries
- Group By & Aggregate Functions
- Window Functions (RANK, cumulative revenue)
- Category-level revenue insights

## Analysis Performed
- Total orders and total revenue
- Highest priced pizza and most common pizza sizes
- Top 5 ordered pizza types
- Quantity ordered per pizza category
- Hour-wise order distribution
- Average pizzas ordered per day
- Top revenue-generating pizzas
- Revenue contribution % by category
- Rank-based analysis using window functions

## Purpose of Project
This project strengthens SQL skills by solving real business questions and generating actionable insights from raw data.
