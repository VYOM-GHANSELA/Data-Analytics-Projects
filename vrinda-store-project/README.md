# Vrinda Store Sales Analysis (Excel & Power BI)

An end-to-end Excel and Power BI project analyzing Vrinda Store’s 2023 sales performance.  
The goal is to identify customer trends and help improve sales strategy for 2024.

## Key Skills Used
- Excel (Cleaning, Pivot Tables, Formulas)
- Power BI (Data Model, DAX Basics, Dashboard)
- Data Visualisation
- Business Insight Generation

## Analysis Performed
- Monthly sales and order comparison
- Gender-wise purchase trends
- State-wise sales contribution
- Age group customer segmentation
- Order status distribution
- Sales channel contribution (Amazon, Flipkart, Myntra)
- Top-selling categories

## Key Insights
- Women customers purchase more than men (≈65%)
- Top contributing states: Maharashtra, Karnataka, Uttar Pradesh
- Age group 30-49 contributes the highest share
- Majority of sales come from Amazon, Flipkart, and Myntra (≈80%)

## Project Purpose
To create a clear, actionable dashboard showing customer behaviour, sales trends, and performance metrics, helping Vrinda Store plan better sales strategies for 2024.
